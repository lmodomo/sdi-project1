Project 1

The theme I select was catching a wave while surfing. I selected this theme since I surfed all the time growing up in Hawaii.

What I programmed was a very simplistic way of catching a wave. There are many other variables like, the location of the other surfers, if there are reef heads sticking up out of the water, if others are in a better position then I, etc...
